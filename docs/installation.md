# Installation

## Stable release

To install Supermetrics client for Python., run this command in your terminal:

```sh
uv add supermetrics
```

Or if you prefer to use `pip`:

```sh
pip install supermetrics
```

## From source

The source files for Supermetrics client for Python. can be downloaded from the [Github repo](https://github.com/Supermetrics/SuperPy-SDK).

You can either clone the public repository:

```sh
git clone git://github.com/Supermetrics/SuperPy-SDK
```

Or download the [tarball](https://github.com/Supermetrics/SuperPy-SDK/tarball/main):

```sh
curl -OJL https://github.com/Supermetrics/SuperPy-SDK/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd supermetrics
uv pip install .
```
